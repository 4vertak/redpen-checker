# Глоссарий проекта «Красная Ручка»

| Термин | Определение |
|--------|-------------|
| **Учитель (Teacher)** | Зарегистрированный пользователь системы, выполняющий проверку работ. |
| **Предмет (Subject)** | Учебная дисциплина (например, русский язык, математика, окружающий мир). |
| **Работа (Work)** | Загруженное изображение тетради, распознанный текст и результаты его проверки. |
| **Эталонный ответ (Expected Answer)** | Правильный ответ или решение, которое учитель может указать для сравнения с работой ученика. |
| **OCR (Optical Character Recognition)** | Процесс распознавания рукописного текста с изображения. Текущая реализация использует GigaChat API. |
| **Анализ текста (Text Analysis)** | Процесс проверки текста на ошибки и формирования пояснений с использованием LLM. |
| **Ошибка (Error)** | Обнаруженное нарушение правила с пояснением причины. |
| **Результат проверки (Check Result)** | Список ошибок, пояснения и предложенная оценка. |
| **Оценка (Grade)** | Итоговая оценка, выставляемая учителем после проверки. |
