# Глоссарий проекта «Красная Ручка»

| Термин | Определение |
|--------|-------------|
| **Пользователь (User)** | Зарегистрированный родитель, использующий систему для проверки работ или получения объяснений. |
| **Предмет (Subject)** | Учебная дисциплина (например, русский язык, математика, окружающий мир). |
| **Работа (Work)** | Загруженное изображение тетради, распознанный текст и результаты его проверки. |
| **Эталонный ответ (Expected Answer)** | Правильный ответ или решение, которое родитель может указать для сравнения с работой ученика. |
| **OCR (Optical Character Recognition)** | Процесс распознавания рукописного текста с изображения. Текущая реализация использует Yandex Vision API. |
| **Анализ текста (Text Analysis)** | Процесс проверки текста на ошибки и формирования пояснений с использованием LLM. |
| **Ошибка (Error)** | Обнаруженное нарушение правила с пояснением причины. |
| **Результат проверки (Check Result)** | Список ошибок и пояснения к ним. |
| **Режим объяснения (Explain Mode)** | Режим, в котором система объясняет, как выполнять задание, без проверки ответа. |
| **Режим проверки (Check Mode)** | Режим, в котором система проверяет ответ ученика на ошибки. |
